# Celebrity Image Classifier using ML

## 📌 Project Overview
This project is a machine learning pipeline designed to accurately identify and classify celebrity faces It processes raw images by isolating critical facial features and converting them to grayscale to standardize inputs and reduce computational noise. The processed images are then compared against an existing dataset to classify faces based on similarity

## 🚀 Key Features
**Targeted Feature Extraction:** Isolates specific facial landmarks, such as the eyes and nose, to improve model focus and accuracy.
**Data Standardization:** Implements grayscale conversion for all input images to streamline processing
**Robust Data Pipeline:** Handles extensive data cleaning and manipulation using Pandas and NumPy to prepare inputs effectively
**Optimized Training:** Utilizes an 80-20 split for training and testing data to ensure rigorous model validation.

## 🛠️ Tech Stack
Language:** Python 
**Data Manipulation:** NumPy, Pandas 
**Machine Learning:** Scikit-learn (for similarity comparison and classification)
**Image Processing:** OpenCV / PIL (Python Imaging Library)
